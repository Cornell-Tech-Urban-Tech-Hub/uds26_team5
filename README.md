# Policy for Housing Equity

Interactive scrollytelling web project exploring how community action becomes policy change through the case of Gowanus, Brooklyn.

## Overview

**Policy for Housing Equity** studies redevelopment as a political process where environmental risk, affordability pressure, and land value collide. Using Gowanus as a case study, the project asks:

- Why do some forms of advocacy remain visible but weak in policy terms?
- Why do other interventions produce enforceable concessions, funding, or oversight?
- How can precedent and historical tactics become a usable tool for present-day organizing?

The project combines neighborhood interviews, case studies, policy scenarios, and spatial storytelling into an interactive web narrative. :contentReference[oaicite:0]{index=0}

## Framework

The project is structured in four parts:

### Insight
Grounding redevelopment in lived experience:
- Resident interviews in Gowanus
- Affordability and displacement pressures
- Environmental justice and uneven implementation

### Transformation
From activism to strategy:
- Community tactics as evidence
- Protest archives as precedent
- Edgemere as a comparative case study

### Prediction
Scenario modeling of tradeoffs:
- Market-led growth, weak protections
- Partial bargaining, partial protections
- Strong protections, better outcomes

### Consensus / NYC Housing Action Atlas
A proposed searchable repository of:
- Past land-use conflicts
- Advocacy tactics
- Policy outcomes
- Reusable strategies for community actors



